乐享拼购推荐码推广活动规则

第一条 适用范围
1.1. 乐享拼购将通过向推广者发放推荐码的方式委托推广者向用户推广乐享拼购的服务/产品，为更好规范乐享拼购(即委托人)、推广者和用户的行为，明确各方的权利义务，并为更好的向广大用户推广乐享拼购的服务/产品，特拟定本规则。推广者通过乐享拼购推荐码推广乐享拼购的服务/产品，以及用户通过推荐码订购服务的，应遵守本规则。

第二条 推荐码的申请
2.1. 推广者提出申请，并经乐享拼购同意的，乐享拼购将向推广者提供推荐码。
2.1.1乐享拼购优先考虑目前在使用vip套餐的用户
2.1.2乐享拼购优先考虑对新零售拼团模式了解的个人、团队、公司
2.1.3乐享拼购优先考虑能对推荐商家提供简易操作培训的个人、团队、公司
2.2. 乐享拼购有权利根据推广者的情况及所提交的申请资料，独立审慎的判断是否通过推广者的推荐码申请。
2.3. 推广者因申请推荐码而向乐享拼购提交的申请材料应当真实、准确。
2.4. 推荐码将发送至推广者申请推荐码的乐享拼购账号。推广者应使用自己的账号向乐享拼购提出推荐码申请，如因借用、租用或因其他账号共用情形引起的纠纷，乐享拼购将不承担任何责任。

第三条 推荐码的推广
3.1. 推广者可以通过推广者有合法使用权利的推广空间或其他合法方式，推广推荐码。
3.2. 推广者应通过正当手段推广乐享拼购推荐码，不采用任何欺骗或虚假性质的行为，不虚假、或扩大陈述乐享拼购的推广政策及优惠。
3.3. 推广者不应擅自修改推荐码，而致使他人将第三方及推广者与乐享拼购的身份或产品/服务相混淆。
3.4. 推广推荐码过程中实施的所有行为均遵守国家法律、法规等规范性文件、本规则及乐享拼购在官网相关页面公布和展示的关于推荐码推广的各项规则的规定和要求，不违背社会公共利益或公共道德，不损害他人的合法权益，不进行任何破坏乐享拼购推荐码推广活动的正常、公平交易秩序、交易规则以及任何损害乐享拼购利益的行为。
3.6. 以上3.2至3.4条所述行为一经出现，乐享拼购将终止与推广者的推广合作，乐享拼购同时有权向推广者追偿给乐享拼购造成的损失，且因此产生的任何法律后果，推广者应以自己的名义独立承担所有的法律责任。

第四条 推荐码的使用
4.1. 通过推广者推广的尚在有效期的推荐码，用户在乐享拼购官网(www.lexiangpingou.cn)首次购买乐享拼购后台任意增值服务时，将享受服务费用9.5折的优惠。

第五条 推广奖励
5.1．使用乐享拼购推荐码销售每一元为一个经验值，完成对应的经验值，系统将自动升级下一个等级！所有的获取的经验值可累计增加，季度增加经验为0将自动降1级直至青铜级别！绑定的标准为新用户在注册时填写的推荐码，他所产生的消费将自动转成对应的经验值！提现可以转入个人微信号或者个人银行卡

5.2． 有效的推广业绩为，通过推广者推广注册时填入推荐码，用户在乐享拼购后台购买的任意功能实际支付金额。
5.3． 就推广者各月的有效推广业绩，乐享拼购将在订制15日之后予以统计，并自动将充值到可提现金额中。
5.4． 提现申请可支持线上微信红包方式发放或者个人银行卡转账，微信红包发放最小金额不能低于1元。

第六条 推广活动的终止
6.1. 不论乐享拼购与推广者之间的推广合作因何原因以何方式终止，推广者了解并同意乐享拼购有权：
① 保存或不保存推广者因申请推荐码、进行推广而提交的材料，如保存的，乐享拼购承诺保密；
② 对于推广者在推广活动终止前实施的违法或违约行为，乐享拼购有追索权。

第七条 其他
7.1. 定义
推广者：即推广者，是指取得乐享拼购账户的，并经乐享拼购同意发放推荐码的单位或个人。
用户：订购乐享拼购官网产品服务的客户。
推广空间：是指推广者合法运营或享有使用权的信息发布空间，包括但不限于网站网页、应用界面和/或即时通讯等软件空间及乐享拼购认可的其他媒体形式。
推荐码：是指乐享拼购通过技术生成的、在乐享拼购平台具有唯一识别性的数字和字母组合。乐享拼购保留对推荐码及展现形式进行自主调整的权利，该调整不以通知推广者为必要。
7.2. 乐享拼购有权随时修改本规则的相关内容，只需公示于乐享拼购官网，并通知推广者，而无需征得推广者的事先同意。修改后的规则应于公示、通知时或公示通知指定的时间即时生效。在乐享拼购修改本规则后，推广者如不同意相关变更的，应停止与乐享拼购的相关合作，否则如继续推广，应被视作已接受了修改后的规则。除非得到乐享拼购的书面授权，任何人将不得修改本规则。
7.3. 本规则的标题仅为方便阅读所设，非对条款的定义、限制、解释或描述其范围或界限。
7.4. 乐享拼购将在乐享拼购网站上公布并不时修订本规则及相关规则（包含隐私权政策），相关规则构成本协议的有效组成部分。
7.5. 乐享拼购有权在法律允许的范围内对本规则做出合理的解释。
7.6. 本规则自发放之日（2016年11月30日）生效。
